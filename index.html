型オープン

伸びてるポストの「構造」だけ借りて、あなたの「型」で書く。X投稿づくりツール。

このツールは無料・無条件で使えます。登録も、リポストも、感想投稿も要りません。
プロンプトを含む全コードは GitHub で公開。 サーバーは存在せず、あなたのAPIキーはブラウザから Anthropic 以外のどこにも送信されません（このページのソースで検証できます）。

1. 設定（初回のみ）

キー未設定 Anthropic Console でAPIキーを発行（要 $5〜 クレジット）。生成1回あたり数円です。

キーとデータはこのブラウザの localStorage にのみ保存されます。共有PCでは使い終わったら「全データ削除」を。

2. あなたの「型」

型 = 口調・一人称・改行のリズム・構成の指示書。手で書くか、過去ポストからAIで抽出できます。プリセット3種入り。
