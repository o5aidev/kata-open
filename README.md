## 隠しているものはありません

この種のツールは普通、プロンプト（AIへの指示文）をサーバー側に隠します。
それが「秘伝のタレ」であり、後段の商材やスクールに誘導する価値の源泉だからです。

このツールは逆をやります。

- **プロンプト全文公開** — `index.html` 内にそのまま書いてあり、アプリ内の「送信されるプロンプトを見る」ボタンでも確認できます。コピーして自分で改造してください
- **サーバーが存在しない** — GitHub Pages の静的ファイル1枚だけ。あなたのAPIキーは、ブラウザから Anthropic のAPIにのみ直接送信されます。「キーをサーバーに保存しません」ではなく、**保存するサーバーがそもそも無い**（Content-Security-Policy で通信先を `api.anthropic.com` に制限しており、ソースを読めば誰でも検証できます）
- **事実を捏造しない設計** — プロンプトで「ナレッジにない実績・数字をでっち上げない」ことをルール化しています。実績が必要な箇所はプレースホルダーになります

## 使い方（3分）

1. **APIキーを用意する（初回のみ）**
   1. [Anthropic Console](https://console.anthropic.com/) でアカウント作成
   2. Billing でクレジットをチャージ（最低 $5。生成1回あたり数円なので数百回分です）
   3. [API Keys](https://console.anthropic.com/settings/keys) で Create Key → `sk-ant-` で始まるキーをコピー
2. **ツールにキーを貼り付けて保存**（このブラウザの localStorage にのみ保存されます）
3. **型を用意する** — プリセット3種入り。自分の過去ポストを貼れば AI が型を抽出します
4. **伸びてるポストを貼り付けて生成** — 構造だけ借りて、あなたの型で最大5案つくります

## 注意事項

- APIキーはパスワードと同じです。人に見せない・SNSに貼らない。漏れたかもと思ったら Console で無効化して再発行
- データ（型・ナレッジ・キー）はブラウザ内にのみあります。キャッシュ削除で消えるので「データ書き出し」でバックアップを（書き出しファイルにキーは含まれません）
- 生成結果は投稿前に必ず自分で確認してください。事実誤認・他者への配慮はあなたの責任です
- 参考にしたポストの**内容の流用はしない**設計ですが、最終判断も投稿者であるあなたにあります
