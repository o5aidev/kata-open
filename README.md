# 軸オープン（jiku-open）

**「何を発信していいかわからない」あなたが、対話で自分の中身に気づき、自分の言葉で書けるようになるツール。**

👉 使う: https://o5aidev.github.io/kata-open/

---

## これは投稿生成ツールではありません

世の中の投稿ツールは「バズった誰かの真似」をさせます。中身が空でも投稿は作れる。だから続かないし、「教えてくれる誰か」への依存が生まれます。

このツールは逆から行きます。

1. **【引き出す】** AIがソクラテス式に1問ずつ質問します。あなたは思い出したまま答えるだけ（音声入力の書き起こしをそのまま貼ってOK）
2. **【軸】** 対話から「誰に・何を・なぜあなたが語れるのか」＝発信の軸と、体験の種（ネタ棚）が貯まっていきます
3. **【書く】** 軸とあなた自身の言葉から投稿を整えます。**案1は常に「あなたの言葉を最小限だけ整えたもの」**——AIの作文ではなく、あなたの言葉が主役です

深度は2つから選べます。**ノーマル**（気軽に）と**本気（丸裸）**——鋭い質問で痛いところまで踏み込みます。すべて、あなたの中身を引き出すためです。

## なぜ無料・無条件なのか

私はAIの使い方を、ある人から無料で教わりました。だから今があります。

このツールは**登録も、リポスト強要も、感想投稿も、何の条件もなし**で使えます。
受け取った人が、いつか自分のやり方で次の誰かに還元してくれたら、それで循環は続きます。

**教わった恩は、次の人へ。**

## 隠しているものはありません

- **プロンプト全文公開** — アプリ内の「送信されるプロンプトを見る」ボタンでいつでも確認できます。コピーして自分で改造してください
- **サーバーが存在しない** — GitHub Pages の静的ファイル1枚。あなたのAPIキーも対話の中身も、ブラウザから Anthropic 以外のどこにも送信されません（Content-Security-Policy で通信先を制限。ソースを読めば誰でも検証できます）
- **捏造しない設計** — あなたが言っていない体験・数字をAIが足すことをプロンプトで禁止しています。投稿前には「これは自分の言葉か？全部本当にあったことか？」の自己チェックが出ます

対話の内容は内省的なものになります。だからこそ**データはこのブラウザの中にしか存在せず、「全データ削除」でいつでも消せます**。

## 使い方（3分）

1. サイトを開いて深度を選ぶ → **1問目はAPIキーなしで始められます**
2. 続きから Anthropic APIキーが必要です:
   [Console](https://console.anthropic.com/) でアカウント作成 → クレジットをチャージ（最低$5・1回の対話は約1円） → [API Keys](https://console.anthropic.com/settings/keys) でキー発行 → ツールに貼って保存
3. あとは対話に答えるだけ。3問ごとに「あなたの軸」が見えてきます

### コストの目安

モデルは Claude Opus 4.8 固定（対話の質を優先）。対話1問 約2〜3円・1セッション（7問+生成）でも30円前後です。Console の Usage limits で月額上限を設定しておくと安心です。

## 注意事項

- APIキーはパスワードと同じです。人に見せない・共有PCでは使い終わったら「全データ削除」
- データはブラウザ内のみ。キャッシュ削除で消えるので「データ書き出し」でバックアップを（書き出しにキーは含まれません）
- 投稿の最終責任はあなたにあります。自己チェック3問（自分の言葉か／言い過ぎていないか／本当にあったことか）を通してから投稿してください
