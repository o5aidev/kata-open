# 軸オープン（jiku-open）

**「何を発信していいかわからない」あなたが、対話で自分の中身に気づき、自分の言葉で書けるようになるツール。**

👉 使う: https://o5aidev.github.io/kata-open/

---

## これは投稿生成ツールではありません

世の中の投稿ツールは「バズった誰かの真似」をさせます。中身が空でも投稿は作れる。だから続かないし、「教えてくれる誰か」への依存が生まれます。

このツールは逆から行きます。

1. **【引き出す】** まず「本当はどうしたいか（北極星）」を一言。あとはAIがソクラテス式に1問ずつ質問し、あなたは思い出したまま答えるだけ（音声入力の書き起こしをそのまま貼ってOK）。裏で判定器が「魂の表裏・系譜・躓き・誰に・立ち位置」の6つの材料を🔴🟡🟢で採点し、次に掘る角度を決めます
2. **【設計書】** 5問（またはコア材料が半分埋まった）時点で、仮のコンセプト設計書が早めに出ます。🔴（まだ設計が立たない）項目はタップで狙って深掘り。🔴が残っていても確定できます——**決めるのはあなた**です
3. **【書く】** 設計書とあなた自身の言葉から投稿を整えます。**案1は常に「あなたの言葉を最小限だけ整えたもの」**——AIの作文ではなく、あなたの言葉が主役です

深度は対話中いつでも切り替えられます。**ノーマル**（気軽に）と**本気（丸裸）**——鋭い質問で痛いところまで踏み込みます。すべて、あなたの中身を引き出すためです。

## なぜ無料・無条件なのか

私はAIの使い方を、ある人から無料で教わりました。だから今があります。

このツールは**登録も、リポスト強要も、感想投稿も、何の条件もなし**で使えます。
受け取った人が、いつか自分のやり方で次の誰かに還元してくれたら、それで循環は続きます。

**教わった恩は、次の人へ。**

## 隠しているものはありません

- **プロンプト全文公開** — アプリ内の「送信されるプロンプトを見る」ボタンで、聞き手・判定器・整文の指示をいつでも確認できます。コピーして自分で改造してください
- **サーバーが存在しない** — GitHub Pages の静的ファイル1枚。あなたのAPIキーも対話の中身も、ブラウザから選んだモデルの会社（Anthropic / OpenAI）以外のどこにも送信されません（Content-Security-Policy で通信先を制限。ソースを読めば誰でも検証できます）
- **捏造しない設計（構造で担保）** — 設計書の材料に使うのは、**あなたが実際に答えた言葉だけ**。AIの質問文を材料にしません。各ターンの出力はJSON形式に固定し、AIが「対話の続き」を勝手に演じて偽の材料を混ぜられない構造にしています。投稿前には「これは自分の言葉か？全部本当にあったことか？」の自己チェックが出ます

対話の内容は内省的なものになります。だからこそ**データはこのブラウザの中にしか存在せず、「全データ削除」でいつでも消せます**。

## 使い方（3分）

1. サイトを開いて「北極星（本当はどうしたいか）」を一言入れて始める → **1問目はAPIキーなしで始められます**
2. 続きから Anthropic APIキーが必要です:
   [Console](https://console.anthropic.com/) でアカウント作成 → クレジットをチャージ（最低$5・1回の対話は約1円） → [API Keys](https://console.anthropic.com/settings/keys) でキー発行 → ツールに貼って保存
3. あとは対話に答えるだけ。5問前後で仮の設計書が出て、🔴の項目を掘っていきます

### モデルは選べます（Claude / GPT）

既定は、聞き手と整文が Claude Opus 4.8（対話の質を優先）、毎ターン走る判定器が Claude Sonnet（コスト抑制）です。
「APIキー」欄でモデルIDを自由に変更できます——候補（claude-opus-4-8 / claude-sonnet-5 / claude-haiku-4-5 / gpt-5.5 / gpt-5.2）から選んでも、任意のモデルIDを直接入力してもOK。**「gpt-」で始まるモデルはOpenAIのAPIキー**（[取得はこちら](https://platform.openai.com/api-keys)）で動きます。聞き手はClaude・判定器はGPT、のような混在もできます。どちらのキーもこのブラウザにのみ保存され、書き出しファイルには含まれません。

### コストの目安

既定構成で対話1問 約2〜3円・1セッション（7問+生成）でも30円前後です。各社Consoleの使用上限（Usage limits）を設定しておくと安心です。

## 注意事項

- APIキーはパスワードと同じです。人に見せない・共有PCでは使い終わったら「全データ削除」
- データはブラウザ内のみ。キャッシュ削除で消えるので「データ書き出し」でバックアップを（書き出しにキーは含まれません）
- 投稿の最終責任はあなたにあります。自己チェック3問（自分の言葉か／言い過ぎていないか／本当にあったことか）を通してから投稿してください
